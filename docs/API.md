## REST Endpoints

### Info & Tokens

#### `GET /api/info`
**Handler:** `getInfo` — `backend/handlers/misc.ts`
**Response:** `{ ip: string, port: number, controlPanelUrl: string }`
Returns the server's LAN IP, port, and control panel origin URL.

#### `GET /api/tokens`
**Handler:** `getTokens` — `backend/handlers/misc.ts`
**Response:** Design tokens JSON (contents of `design/tokens.json`). Cached after first read.

#### `POST /api/sync-start`
**Handler:** `syncStart`
**Body:** _(none)_
**Response:** `{ ts_abs: number }`
**Emits:** `sync_start { ts_abs }`
Sets the sidecar logger's relative timestamp origin.

### Scene & Focus

#### `GET /api/scene`
**Handler:** `getScene`
**Response:** `{ title: string, subtitle: string, visible: boolean }` — in-memory, resets on server restart.

#### `POST /api/scene/change`
**Handler:** `changeScene`
**Body:** `{ title: string, subtitle?: string, visible?: boolean }`
**Response:** scene state
**Emits:** `scene_changed { title, subtitle, visible }`

#### `POST /api/character-focus`
**Handler:** `focusCharacter`
**Body:** `{ charId: string }` — pass `""` to unfocus
**Response:** `{ focused: Character | null }`
**Emits:** `character_focused { character }` or `character_unfocused {}`

### Characters

#### `GET /api/characters`
**Handler:** `listCharacters`
**Response:** `Character[]`

#### `GET /api/characters/:id`
**Handler:** `getCharacter`
**Response:** `Character` — **404** if not found.

#### `POST /api/characters`
**Handler:** `createCharacter`
**Body:**
```json
{
  "name": "string (required)",
  "player": "string (required)",
  "hp_max": "number > 0 (required)",
  "hp_current": "number >= 0 (optional, defaults to hp_max)",
  "species": "string?",
  "class_name": "string?",
  "subclass_name": "string?",
  "level": "number?",
  "hp_temp": "number?",
  "ac_base": "number?",
  "speed": "number?",
  "proficiency_bonus": "number?",
  "ability_scores": "object?",
  "saving_throws_proficiencies": "object?",
  "skill_proficiencies": "string[]?",
  "expertise": "string[]?",
  "conditions": "Condition[]?",
  "resources": "ResourceSlot[]?",
  "is_active": "boolean?",
  "is_visible_to_party_overlay": "boolean?",
  "portrait": "string?",
  "notes": "any[]?"
}
```
**Response:** `Character` (201)
**Emits:** `character_created { character }`

#### `POST /api/characters/batch/hp`
**Handler:** `batchUpdateHp`
**Body:** `{ updates: Array<{ charId: string, hp_current: number }> }`
**Response:** `{ results: Character[], errors: Array<{ charId, error }> }`
**Emits:** `hp_updated { character, hp_current }` per successful update.

#### `PUT /api/characters/:id/hp`
**Handler:** `updateHp`
**Body:** `{ hp_current: number }`
**Response:** `Character`
**Emits:** `hp_updated { character, hp_current }`

#### `PUT /api/characters/:id/photo`
**Handler:** `updatePhoto`
**Body:** `{ portrait: string }` — URL or base64. Max 2 MB.
**Response:** `Character`
**Emits:** `character_updated { character }`

#### `PUT /api/characters/:id`
**Handler:** `updateCharacter`
**Body:** Any subset of character fields (all optional): `name`, `player`, `species`, `class_name`, `subclass_name`, `level`, `hp_max`, `hp_current`, `hp_temp`, `ac_base`, `speed`, `proficiency_bonus`, `ability_scores`, `saving_throws_proficiencies`, `skill_proficiencies`, `expertise`, `is_active`, `is_visible_to_party_overlay`, `notes`.
**Response:** `Character`
**Emits:** `character_updated { character }`

#### `DELETE /api/characters/:id`
**Handler:** `deleteCharacter`
**Response:** `{ ok: true }`
**Emits:** `character_deleted { charId }`

#### `POST /api/characters/:id/conditions`
**Handler:** `addCondition`
**Body:** `{ condition_name: string, intensity_level?: number }`
**Response:** `Condition` (201)
**Emits:** `condition_added { charId, condition }`

#### `DELETE /api/characters/:id/conditions/:condId`
**Handler:** `removeCondition`
`condId` must be a 5-char alphanumeric ID.
**Response:** `{ ok: true }`
**Emits:** `condition_removed { charId, conditionId }`

#### `PUT /api/characters/:id/resources/:rid`
**Handler:** `updateResource`
**Body:** `{ pool_current: number }` — clamped to `[0, pool_max]`.
**Response:** updated `ResourceSlot`
**Emits:** `resource_updated { charId, resource }`

#### `POST /api/characters/:id/rest`
**Handler:** `restoreResources`
**Body:** `{ type: "short" | "long" }`
Short rest restores `reset_on: "short_rest"` resources only. Long rest restores both `short_rest` and `long_rest`.
**Response:** `{ restored: string[] }` — names of restored resources
**Emits:** `rest_taken { charId, type, restored, character }`

### Encounter

#### `GET /api/encounter`
**Handler:** `getEncounter`
**Response:** `{ active: boolean, round: number, currentTurnIndex: number, participants: Participant[] }`
In-memory only — resets on server restart.

#### `POST /api/encounter/start`
**Handler:** `startEncounter`
**Body:** `{ participants: Array<{ charId: string, initiative: number }> }`
Participants sorted by initiative descending. Unknown `charId`s silently dropped.
**Response:** encounter state
**Emits:** `encounter_started { active, round, currentTurnIndex, participants }`

#### `POST /api/encounter/next-turn`
**Handler:** `nextTurn`
**Body:** _(none)_
Advances `currentTurnIndex`. Wraps to 0 and increments `round` at end of initiative order.
**Response:** encounter state
**Emits:** `turn_advanced { currentTurnIndex, currentParticipant, round }`

#### `POST /api/encounter/end`
**Handler:** `endEncounter`
**Body:** _(none)_
**Response:** `{ ok: true }`
**Emits:** `encounter_ended {}`

### `errors.ts`

`control-panel/src/lib/services/errors.ts`

```typescript
type ServiceErrorCode = 'CONFIG' | 'VALIDATION' | 'NOT_FOUND' | 'UNAUTHORIZED' | 'NETWORK' | 'UNKNOWN'

class ServiceError extends Error {
  code: ServiceErrorCode
  context?: Record<string, unknown>   // operation, id, collection, etc.
  cause?: unknown                     // original PocketBase/transport error
}
```

**PocketBase → ServiceError mapping:**

| PocketBase status | ServiceError code |
|---|---|
| 400, 422 | `VALIDATION` |
| 404 | `NOT_FOUND` |
| 401, 403 | `UNAUTHORIZED` |
| `TypeError` (fetch) | `NETWORK` |
| Other | `UNKNOWN` |
