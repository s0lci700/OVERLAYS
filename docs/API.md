# API Reference

> Auto-generated by /doc-steward api. Edit source files, not this doc.

## Frontend Services

### `errors.ts` ÔÇö `control-panel/src/lib/services/errors.ts`

**`ServiceError`** (class)
Normalized error shape for all service failures. Extends `Error`.
- `code: ServiceErrorCode` ÔÇö `'CONFIG' | 'VALIDATION' | 'NOT_FOUND' | 'UNAUTHORIZED' | 'NETWORK' | 'UNKNOWN'`
- `message: string` ÔÇö human-readable summary
- `context?: Record<string, unknown>` ÔÇö debugging metadata
- `cause?: unknown` ÔÇö original thrown error

Catch with `instanceof ServiceError` and branch on `err.code`.

---

### `pocketbase.ts` ÔÇö `control-panel/src/lib/services/pocketbase.ts`

**`pb`** (singleton) ÔÇö `PocketBase` instance initialized from `VITE_POCKETBASE_URL`. Shared across all service modules.

**`getCharacterRecord(id: string)`** Ôćĺ `Promise<CharacterRecord>`
Fetches one character from the `characters` collection. Throws `ServiceError('NOT_FOUND')` on 404.

**`updateCharacterRecord(id: string, data: Partial<CharacterRecord>)`** Ôćĺ `Promise<CharacterRecord>`
Partial update on a character record. Throws `ServiceError('VALIDATION')` on 400/422.

**`listCharacterRecords()`** Ôćĺ `Promise<CharacterRecord[]>`
Fetches all characters from the `characters` collection.

**`listActiveCharacters()`** Ôćĺ `Promise<CharacterRecord[]>`
Fetches only characters where `is_active = true`.

**`addCondition(characterId: string, condition: { condition_name: string, intensity_level?: number })`** Ôćĺ `Promise<CharacterRecord>`
Adds a condition to a character's conditions array.

**`removeCondition(characterId: string, conditionId: string)`** Ôćĺ `Promise<CharacterRecord>`
Removes a condition by ID from a character's conditions array.

**`updateResource(characterId: string, resourceId: string, poolCurrent: number)`** Ôćĺ `Promise<CharacterRecord>`
Updates a resource's `pool_current` value.

**`restoreResources(characterId: string, restType: 'short' | 'long')`** Ôćĺ `Promise<CharacterRecord>`
Restores resources matching the given rest type.

**`getPortraitUrl(record: CharacterRecord)`** Ôćĺ `string`
Returns the full-size portrait file URL from a PocketBase record.

**`getPortraitThumbUrl(record: CharacterRecord, size?: string)`** Ôćĺ `string`
Returns a thumbnail portrait URL at the specified size (default `100x100`).

All functions throw typed `ServiceError` for all PocketBase and network failures.

### `socket.ts` ÔÇö `control-panel/src/lib/services/socket.ts`

Typed Socket.io client. Singleton pattern ÔÇö one socket shared across the control panel.

**`socketStatus`** ÔÇö `$state({ connected: boolean, lastSync: Date | null })`
Reactive connection state. Updates on `connect` / `disconnect` events.

**`connectSocket(sessionId: string, surface: string)`** Ôćĺ `void`
Creates socket with query params `{ sessionId, surface }` and connects. No-op if already connected.

**`disconnectSocket()`** Ôćĺ `void`
Disconnects the socket. Warns if not initialized.

**`getSocket()`** Ôćĺ `Socket`
Returns the singleton socket. Throws `ServiceError('CONFIG')` if not yet initialized.

**`subscribe<K extends keyof EventPayloadMap>(event: K, handler)`** Ôćĺ `() => void`
Typed subscription. Returns an unsubscribe function. Throws `ServiceError('CONFIG')` if socket not initialized.

**`emit<K extends keyof EventPayloadMap>(event: K, payload)`** Ôćĺ `void`
Typed emit. Throws `ServiceError('CONFIG')` if socket not connected.

Event types are constrained to `EventPayloadMap` from `$lib/contracts/events.ts`.

## Error Handling Pattern

All service calls follow the same pattern:

```typescript
try {
  const char = await getCharacterRecord(id);
} catch (err) {
  if (err instanceof ServiceError) {
    if (err.code === 'NOT_FOUND') navigate('/characters');
    if (err.code === 'VALIDATION') showFormError(err.message);
    if (err.code === 'UNAUTHORIZED') promptLogin();
    if (err.code === 'NETWORK') showOfflineBanner();
  }
}
```
