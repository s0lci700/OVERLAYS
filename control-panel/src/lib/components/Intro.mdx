# Dados & Risas Component Surfaces

Components are organized by the physical or digital **surface** where they appear:

- **Stage**: Full-screen production displays (e.g. wallboards).
- **Cast**: Interfaces for the DM and Players (tablets, phones).
- **Commons**: Passive displays for everyone in the room (initiative, status board).
- **Audience**: OBS overlays for the Twitch/YouTube stream.
- **Shared**: Generic UI components (buttons, dialogs, form inputs).
